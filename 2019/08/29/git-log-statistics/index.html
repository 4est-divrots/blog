Gitでコミット数、変更行数、ファイル数を取得する色々

複数ユーザがいるrepositoryで自分の書いたコードがどれだけ占めるかを知りたい時に。

ユーザーごとのコミット数

git shortlogを使えば一発

返り値

説明
-n コミット数降順に並べる
-s コミットメッセージを省略して一行にする
-e メールアドレスも表示する

authorを指定して合算を取得する
